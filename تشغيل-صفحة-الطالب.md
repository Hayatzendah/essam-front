# كيف تشغّل صفحة الطالب وتشوف التعديلات

## ليش ما يظهر أي تغيير؟

التعديلات موجودة في **الكود** (مجلد `essam-front`).  
التغيير يظهر فقط لما المتصفح يفتح **نفس هذا المشروع** وهو شغال من عندك.

- لو بفتح الموقع من **رابط منشور** (مثلاً على سيرفر أو دومين): المتصفح يحمّل نسخة قديمة من الموقع، فما بتحصل على التعديلات.
- لو بفتح **localhost** لكن من مشروع تاني أو من مجلد مختلف: الكود اللي شغال مش نفس اللي تم تعديله.

## الخطوات (بالترتيب)

### 1) التأكد إنك داخل مجلد الواجهة

من الطرفية (PowerShell أو CMD):

```bash
cd f:\essam-quiz\essam-front
```

(أو المسار الصح لمشروعك لو مختلف.)

### 2) تشغيل الواجهة (مع مسح كاش Vite)

لو عدّلت على الـ CSS أو الصفحة وما ظهر التغيير، جرّب:

```bash
npm run dev:fresh
```

أو أوقف السيرفر (Ctrl+C) وامسح كاش Vite ثم شغّل عادي:

- **PowerShell:** `Remove-Item -Recurse -Force node_modules\.vite -ErrorAction SilentlyContinue; npm run dev`
- **CMD:** `rd /s /q node_modules\.vite` ثم `npm run dev`

استنى لحد ما يطلع رسالة مثل:

```
  VITE v5.x.x  ready in xxx ms
  ➜  Local:   http://localhost:5173/
```

### 3) فتح الرابط من المتصفح

- افتح **نفس الرابط اللي ظهر عندك** (مثلاً `http://localhost:5173` أو `http://localhost:5174`).
- **مهم:** لا تفتح رابط منشور (مثل دومين خارجي)، افتح **localhost** من الطرفية.

### 4) مسح الكاش عشان تتأكد إنك شايف آخر نسخة

بعد ما الصفحة تفتح:

- **Windows:** `Ctrl + Shift + R` أو `Ctrl + F5`
- أو: F12 → كليك يمين على زر التحديث → **Empty Cache and Hard Reload**

### 5) فتح امتحان كطالب

من نفس الموقع (localhost) ادخل على صفحة الطالب وافتح أي امتحان فيه فقرة قراءة أو أسئلة (Frage / Antwort).

### 6) التأكد إن الكود الجديد شغال

افتح أدوات المطوّر (F12) → تبويب **Elements** (أو العناصر) → ابحث (Ctrl+F) عن:  
`data-exam-css-version`  
لو ظهر عندك عنصر فيه `data-exam-css-version="text-fix-v1"` معناها الصفحة محمّلة من الكود الجديد.  
لو ما ظهر هالسمة، السيرفر لسا يخدم نسخة قديمة أو الكاش لسا قديم.

---

## لو لسا ما في تغيير

1. **تأكد إنك فعلاً بفتح localhost** (الشريط فيه `localhost:5173` أو `5174`).
2. جرّب نافذة **خاصة / Incognito** وافتح نفس الرابط.
3. لو عندك موقع منشور وتريد التعديلات تظهر هناك: اعمل **Build** ثم انشر الملفات الجديدة:
   ```bash
   cd f:\essam-quiz\essam-front
   npm run build
   ```
   بعدين ارفع محتويات مجلد `dist` للسيرفر/الاستضافة اللي يستخدمها الموقع المنشور.
